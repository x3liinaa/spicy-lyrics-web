# Spicy-Lyrics Web

A clean, web-based lyrics visualizer for Spotify. This project is a standalone web port inspired by the Spicy Lyrics extension for Spicetify.

## Project Origin
Developed in a quick one night vibe code session so nothing too serious

## Usage
1. Open the website.
2. Connect your Spotify account.
3. Play music on any device to see live lyrics.

## Disclaimer
This is an unofficial project and is not affiliated with Spotify or the original Spicetify developers.
